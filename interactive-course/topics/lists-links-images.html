Types of Lists

HTML provides three types of lists: unordered lists (bullet points), ordered lists (numbered), and description lists (term-definition pairs).

Hyperlinks

The Power of Links

Hyperlinks are what make the web "web-like." They connect pages, sections, and resources together, allowing users to navigate seamlessly.

External Links

<a href="https://www.google.com" target="_blank">
    Visit Google
</a>

Links to external websites. Use target="_blank" to open in new tab.

Internal Links

<a href="about.html">About Us</a>
<a href="../index.html">Home</a>

Links to other pages within your website.

Anchor Links

<a href="#section1">Go to Section 1</a>
<h2 id="section1">Section 1</h2>

Links to specific sections within the same page.

Email Links

<a href="mailto:contact@example.com">
    Send Email
</a>

Links that open the user's email client.

Images

Adding Visual Content

Images make web pages more engaging and informative. The <img> element is self-closing and requires the src attribute.

Image Syntax

<img src="image.jpg" alt="Description of image" width="300" height="200">

src

Specifies the path to the image file (required)

alt

Alternative text for accessibility and SEO (required)

width

Sets the width of the image in pixels

height

Sets the height of the image in pixels

Image Types and Sources

Local Images

<img src="images/logo.png" alt="Company Logo">

Images stored in your website's folder

External Images

<img src="https://via.placeholder.com/300x200" alt="Placeholder">

Images hosted on other servers
