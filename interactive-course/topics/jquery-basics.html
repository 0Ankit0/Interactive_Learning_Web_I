What is jQuery?

jQuery is a fast, lightweight, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across different browsers.

Vanilla JavaScript

const elements = document.querySelectorAll('.btn');
for (let i = 0; i < elements.length; i++) {
    elements[i].addEventListener('click', function() {
        this.style.backgroundColor = 'blue';
    });
}
8 lines of code
VS

jQuery

$(document).ready(function() {
    $('.btn').click(function() {
        $(this).css('background-color', 'blue');
    });
});
