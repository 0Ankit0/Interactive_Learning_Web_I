History of the Web

The NeXT computer used by Tim Berners-Lee at CERN - the world's first web server

Key Insight

The World Wide Web was invented in 1989 by Tim Berners-Lee at CERN, initially as a solution to share information between researchers worldwide.

The history of the web begins with Tim Berners-Lee's frustration with the inefficiencies of finding information stored on different computers at CERN. On March 12, 1989, he submitted a proposal titled "Information Management: A Proposal" that would change the world forever.

Versions of the Web

Web 1.0 (1989-2004)

Read-Only Web

Characteristics:

  • Static HTML pages
  • One-way information flow
  • Limited interactivity
  • Dial-up connections
Example: Early news websites, company brochures

Web 2.0 (2004-2016)

Read-Write Web

Characteristics:

  • User-generated content
  • Social media platforms
  • AJAX and dynamic content
  • Collaboration and sharing
Example: Facebook, YouTube, Wikipedia, blogs

Web 3.0 (2016-present)

Semantic Web

Characteristics:

  • AI and machine learning
  • Personalized experiences
  • IoT integration
  • Blockchain technology
Example: Voice assistants, smart recommendations

Web 4.0 (Future)

Ubiquitous Web

Characteristics:

  • Artificial intelligence
  • Ambient computing
  • Virtual/Augmented reality
  • Brain-computer interfaces
Example: Metaverse, AI assistants, smart cities
