What is the World Wide Web?

Definition

The World Wide Web (WWW) is a system of interconnected documents and resources linked by hyperlinks and URLs, accessible via the Internet. It consists of web pages, websites, and web applications that can be viewed using web browsers.

Key Terms

  • World Wide Web (WWW): A collection of interconnected documents and resources
  • Hyperlink: A reference to another document or resource
  • URL (Uniform Resource Locator): The address of a web page or resource
  • Web Browser: Software used to access and view web pages
  • Website: A collection of related web pages

History of the Web

The NeXT computer used by Tim Berners-Lee at CERN - the world's first web server

Key Insight

The World Wide Web was invented in 1989 by Tim Berners-Lee at CERN, initially as a solution to share information between researchers worldwide.

Why It Matters

The web started as a way for scientists to share ideas easily. Today, it connects billions of people worldwide, like how a phone lets you call friends—except it's for information and fun!

The history of the web begins with Tim Berners-Lee's frustration with the inefficiencies of finding information stored on different computers at CERN. On March 12, 1989, he submitted a proposal titled "Information Management: A Proposal" that would change the world forever.

Key Milestones in Web History

1980

ENQUIRE

Tim Berners-Lee creates ENQUIRE, a hypertext system that serves as a precursor to the Web.

1989

Web Proposal

Berners-Lee submits "Information Management: A Proposal" at CERN, proposing a "web" of hypertext documents.

1990

WWW Components

Development of HTTP, HTML, URLs, and the first web browser (WorldWideWeb) begins.

1991

First Website

The first website goes online at info.cern.ch, explaining the World Wide Web project.

URLs and Addressing

Did You Know?

The first URL ever created was http://info.cern.ch/hypertext/WWW/TheProject.html, and it's still accessible today!

Uniform Resource Locators (URLs) provide a way to identify and locate resources on the web. Understanding URL structure is fundamental to web development.

Anatomy of a URL

https://www.example.com:443/path/to/page?id=123&name=test#section

Step-by-Step: How URLs Work

  1. Protocol: Like choosing "call" or "text" on your phone (e.g., https:// means secure).
  2. Domain: The website's name, like a house address (e.g., google.com).
  3. Path: The specific page, like a room in the house (e.g., /search).
  4. Query: Extra details, like asking for a specific item (e.g., ?q=web).

Versions of the Web

Web 1.0 (1989-2004)

Read-Only Web

Characteristics:

  • Static HTML pages
  • One-way information flow
  • Limited interactivity
  • Dial-up connections
Example: Early news websites, company brochures

Web 2.0 (2004-2016)

Read-Write Web

Characteristics:

  • User-generated content
  • Social media platforms
  • AJAX and dynamic content
  • Collaboration and sharing
Example: Facebook, YouTube, Wikipedia, blogs

Web 3.0 (2016-present)

Semantic Web

Characteristics:

  • AI and machine learning
  • Personalized experiences
  • IoT integration
  • Blockchain technology
Example: Voice assistants, smart recommendations

Web 4.0 (Future)

Ubiquitous Web

Characteristics:

  • Artificial intelligence
  • Ambient computing
  • Virtual/Augmented reality
  • Brain-computer interfaces
Example: Metaverse, AI assistants, smart cities

Think of It Like This

Web 1.0 is like a library book—you read it but can't write notes. Web 2.0 is like a shared notebook where everyone adds ideas. Web 3.0 is like a smart assistant that knows what you like and suggests things.
