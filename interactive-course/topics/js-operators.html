Introduction to JavaScript Operators

What are Operators?

Operators are special symbols in JavaScript that perform operations on operands (values and variables). They are the building blocks of expressions and allow you to manipulate data, perform calculations, make comparisons, and control program flow.

What are Expressions?

An expression is any valid unit of code that resolves to a value. The simplest expressions are literals (like numbers or strings), but most expressions involve operators that combine operands to produce new values.

Key Concepts

Operands: Values that operators act upon
Binary Operators: Work with two operands (e.g., 5 + 3)
Unary Operators: Work with one operand (e.g., -x, ++y)
Ternary Operator: Works with three operands (condition ? true : false)

Types of JavaScript Operators

JavaScript provides various types of operators to perform different operations on data.

Arithmetic Operators

Perform mathematical calculations on numeric values.

let a = 10, b = 3;

console.log(a + b);    // 13 (Addition)
console.log(a - b);    // 7 (Subtraction)
console.log(a * b);    // 30 (Multiplication)
console.log(a / b);    // 3.33... (Division)
console.log(a % b);    // 1 (Modulus/Remainder)
console.log(a ** b);   // 1000 (Exponentiation)

// Increment/Decrement
console.log(++a);      // 11 (Pre-increment)
console.log(b--);      // 3 (Post-decrement)

Assignment Operators

Assign values to variables, often with operations.

let x = 5;        // Basic assignment

x += 3;           // x = x + 3  → 8
x -= 2;           // x = x - 2  → 6
x *= 4;           // x = x * 4  → 24
x /= 3;           // x = x / 3  → 8
x %= 5;           // x = x % 5  → 3
x **= 2;          // x = x ** 2 → 9

// Multiple assignment
let a, b, c;
a = b = c = 10;   // All equal 10

Comparison Operators

Compare values and return boolean results.

let a = 5, b = "5", c = 10;

console.log(a == b);   // true (loose equality)
console.log(a === b);  // false (strict equality)
console.log(a != c);   // true (not equal)
console.log(a !== b);  // true (strict not equal)
console.log(a < c);    // true (less than)
console.log(a <= b);   // true (less than or equal)
console.log(c > a);    // true (greater than)
console.log(c >= a);   // true (greater than or equal)

Logical Operators

Combine or modify boolean values.

let a = true, b = false, c = true;

// AND operator (&&)
console.log(a && c);    // true
console.log(a && b);    // false

// OR operator (||)
console.log(a || b);    // true
console.log(b || false); // false

// NOT operator (!)
console.log(!a);        // false
console.log(!b);        // true
console.log(!(a && c)); // false

Unary Operators

Operate on a single operand.

let x = 5;
let str = "123";
let obj = { name: "John" };

// typeof operator
console.log(typeof x);     // "number"
console.log(typeof str);   // "string"

// Unary plus/minus
console.log(+str);         // 123 (string to number)
console.log(-x);           // -5 (negation)

// delete operator
delete obj.name;
console.log(obj.name);     // undefined

Ternary Operator

Conditional operator with three operands.

// Syntax: condition ? valueIfTrue : valueIfFalse

let age = 18;
let status = age >= 18 ? "adult" : "minor";
console.log(status);        // "adult"

let score = 85;
let grade = score >= 90 ? "A" : 
            score >= 80 ? "B" : 
            score >= 70 ? "C" : "F";
console.log(grade);         // "B"

// Nested ternary
let weather = "sunny";
let activity = weather === "sunny" ? "beach" : 
               weather === "rainy" ? "movies" : "park";

Precedence Example:

let result = 2 + 3 * 4;        // 14 (not 20)
// Multiplication has higher precedence than addition
// So it's evaluated as: 2 + (3 * 4)

let withParens = (2 + 3) * 4;  // 20
// Parentheses have highest precedence

let complex = 5 + 2 * 3 ** 2;  // 23
// Order: 3**2 = 9, then 2*9 = 18, then 5+18 = 23
