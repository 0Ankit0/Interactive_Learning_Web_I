Modern Layout Systems

CSS Flexbox

  • One-dimensional layout (row or column)
  • Perfect for component-level layouts
  • Great for aligning items
  • Flexible item sizing

Best For:

Navigation bars, form controls, card layouts, centering content

CSS Grid

  • Two-dimensional layout (rows and columns)
  • Perfect for page-level layouts
  • Complex grid structures
  • Precise positioning control

Best For:

Page layouts, dashboards, image galleries, complex designs

Flexbox Properties Reference

display: flex

Defines a flex container

.container {
  display: flex;
}

flex-direction

Defines the main axis direction

flex-direction: row | row-reverse | column | column-reverse;

justify-content

Aligns items along main axis

justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;

align-items

Aligns items along cross axis

align-items: stretch | flex-start | flex-end | center | baseline;

flex-grow

How much an item should grow

.item {
  flex-grow: 0; /* default */
  flex-grow: 1; /* grow to fill space */
}

flex-shrink

How much an item should shrink

.item {
  flex-shrink: 1; /* default */
  flex-shrink: 0; /* don't shrink */
}

flex-basis

Initial size before growing/shrinking

.item {
  flex-basis: auto; /* default */
  flex-basis: 200px;
  flex-basis: 20%;
}

align-self

Override align-items for individual item

.item {
  align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
