What are Regular Expressions?

Regular expressions (regex or regexp) are patterns used to match character combinations in strings. In JavaScript, they are objects that can be used with various string methods to search, replace, and validate text.

Search & Match

Find specific patterns within text, like email addresses, phone numbers, or URLs. 

const text = "Call me at 555-1234";
const phonePattern = /\d{3}-\d{4}/;
console.log(text.match(phonePattern)); // ["555-1234"]

Replace & Substitute

Replace patterns in text with other content, perfect for formatting and cleaning data. 

const text = "Hello World!";
const newText = text.replace(/World/, "JavaScript");
console.log(newText); // "Hello JavaScript!"

Validate Input

Check if user input matches expected formats, essential for form validation. 

const email = "user@example.com";
const emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
console.log(emailPattern.test(email)); // true

Split & Parse

Break text into arrays based on complex patterns, useful for parsing CSV or logs. 

const csv = "name,age,city";
const fields = csv.split(/,/);
console.log(fields); // ["name", "age", "city"]

Basic Regex Syntax

Regular expressions in JavaScript can be created in two ways:

Literal Notation

// Using forward slashes
const regex1 = /pattern/flags;
const regex2 = /hello/i;  // Case-insensitive
Best for: Static patterns known at compile time

Constructor Function

// Using RegExp constructor
const regex1 = new RegExp("pattern", "flags");
const regex2 = new RegExp("hello", "i");
Best for: Dynamic patterns built at runtime

Character Classes & Metacharacters

.
Matches any character except newline
a.c matches "abc", "a1c", "a@c"
\
Escape character for special characters
\. matches literal "." character
|
Alternation (OR operator)
cat|dog matches "cat" or "dog"
()
Grouping and capturing
(abc)+ matches "abc", "abcabc"
[]
Character set (any character inside)
[abc] matches "a", "b", or "c"
[^]
Negated character set
[^abc] matches any char except a, b, c
\d
Any digit (0-9)
\d{3} matches "123", "456"
\D
Any non-digit
\D+ matches "abc", "hello"
\w
Word character (a-z, A-Z, 0-9, _)
\w+ matches "word", "test123"
\W
Non-word character
\W+ matches "!@#", " "
\s
Whitespace (space, tab, newline)
\s+ matches spaces, tabs
\S
Non-whitespace character
\S+ matches "word", "123"
*
Zero or more occurrences
ab* matches "a", "ab", "abbb"
+
One or more occurrences
ab+ matches "ab", "abbb" (not "a")
?
Zero or one occurrence (optional)
colou?r matches "color", "colour"
{n}
Exactly n occurrences
\d{3} matches exactly 3 digits
{n,}
n or more occurrences
\d{3,} matches 3 or more digits
{n,m}
Between n and m occurrences
\d{3,5} matches 3 to 5 digits
^
Start of string/line
^Hello matches "Hello" at start
$
End of string/line
end$ matches "end" at end
\b
Word boundary
\bcat\b matches "cat" as whole word
\B
Non-word boundary
\Bcat\B matches "cat" inside words

JavaScript Regex Methods

JavaScript provides several methods for working with regular expressions:

test()

Returns: boolean

Tests if a pattern matches in a string. Returns true or false. 

const regex = /hello/i;
console.log(regex.test("Hello World")); // true
console.log(regex.test("Hi there"));    // false
Result: true

match()

Returns: Array | null

Retrieves the matches when matching a string against a regex. 

const text = "The year 2023 was great!";
const matches = text.match(/\d+/g);
console.log(matches); // ["2023"]
Result: ["5", "3"]

replace()

Returns: string

Returns a new string with matches replaced by a replacement string. 

const text = "Hello World";
const newText = text.replace(/World/, "JavaScript");
console.log(newText); // "Hello JavaScript"
Result: "Hello JavaScript! JavaScript is great!"

search()

Returns: number

Returns the index of the first match, or -1 if not found. 

const text = "Find the number 42 here";
const index = text.search(/\d+/);
console.log(index); // 16
Result: 14

split()

Returns: Array

Splits a string into an array using a regex as the delimiter. 

const text = "apple,banana;orange:grape";
const fruits = text.split(/[,;:]/);
console.log(fruits); // ["apple", "banana", "orange", "grape"]
Result: ["apple", "banana", "orange", "grape"]

exec()

Returns: Array | null

Executes a search for a match and returns detailed information. 

const regex = /(\d{4})-(\d{2})-(\d{2})/;
const result = regex.exec("Today is 2023-12-25");
console.log(result); // ["2023-12-25", "2023", "12", "25", ...]
Result: ["2023-12-25", "2023", "12", "25"]

Common Regex Patterns

Here are some frequently used regex patterns for common validation tasks:

Email Address
/^[^\s@]+@[^\s@]+\.[^\s@]+$/
Validates basic email format: no spaces, @ symbol, domain with dot
✓ user@example.com ✓ test.email@domain.org ✗ invalid.email ✗ @domain.com
Phone Number (US)
/^(\+1[-.\s]?)?\(?([0-9]{3})\)?[-.\s]?([0-9]{3})[-.\s]?([0-9]{4})$/
Matches US phone numbers in various formats
✓ (555) 123-4567 ✓ 555.123.4567 ✓ +1 555 123 4567 ✗ 123-456
Strong Password
/^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{8,}$/
At least 8 chars: lowercase, uppercase, digit, special character
✓ MyPass123! ✓ Secure$456 ✗ password ✗ 123456
URL
/^https?:\/\/(?:[-\w.])+(?:[:\d]+)?(?:\/(?:[\w/_.])*)?(?:\?(?:[\w&=%.])*)?(?:#(?:[\w.])*)?$/
Matches HTTP and HTTPS URLs with optional ports, paths, queries
✓ https://example.com ✓ http://site.com:8080/path ✗ ftp://example.com ✗ not-a-url
Date Extraction (YYYY-MM-DD)
/(\d{4})-(\d{2})-(\d{2})/g
Extracts dates in ISO format with captured groups for year, month, day
Test: "The events are on 2023-12-25 and 2024-01-15"
Hashtags
/#\w+/g
Finds all hashtags in text (# followed by word characters)
Test: "Learning #JavaScript and #RegEx is fun! #coding"
Currency Amounts
/\$\d{1,3}(?:,\d{3})*(?:\.\d{2})?/g
Matches dollar amounts with proper comma separation and optional cents
Test: "Prices: $29.99, $1,234.56, $10,000"
HTML Tags
/<([a-z]+)(\s[^>]*)?>.*?<\/\1>/gi
Matches opening and closing HTML tags with content
Test: "<div class='test'>Content</div> <p>Paragraph</p>"
Title Case
/\b\w+/g with replacement function
Converts text to title case (first letter of each word capitalized)
Hello World Javascript
Remove Extra Spaces
/\s+/g replace with single space
Removes multiple consecutive spaces, tabs, newlines
text with extra spaces
Format Phone Numbers
/(\d{3})(\d{3})(\d{4})/ → ($1) $2-$3
Formats 10-digit numbers into (XXX) XXX-XXXX format
(123) 456-7890
Mask Sensitive Data
/(\d{4})\d{4}(\d{4})/ → $1****$2
Masks middle digits of credit card numbers
1234****3456
