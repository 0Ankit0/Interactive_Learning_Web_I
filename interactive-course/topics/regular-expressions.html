What are Regular Expressions?

Regular expressions (regex or regexp) are patterns used to match character combinations in strings. In JavaScript, they are objects that can be used with various string methods to search, replace, and validate text.

Search & Match

Find specific patterns within text, like email addresses, phone numbers, or URLs. 

const text = "Call me at 555-1234";
const phonePattern = /\d{3}-\d{4}/;
console.log(text.match(phonePattern)); // ["555-1234"]

Replace & Substitute

Replace patterns in text with other content, perfect for formatting and cleaning data. 

const text = "Hello World!";
const newText = text.replace(/World/, "JavaScript");
console.log(newText); // "Hello JavaScript!"

Validate Input

Check if user input matches expected formats, essential for form validation. 

const email = "user@example.com";
const emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
console.log(emailPattern.test(email)); // true

Split & Parse

Break text into arrays based on complex patterns, useful for parsing CSV or logs. 

const csv = "name,age,city";
const fields = csv.split(/,/);
console.log(fields); // ["name", "age", "city"]

Basic Regex Syntax

Regular expressions in JavaScript can be created in two ways:

Literal Notation

// Using forward slashes
const regex1 = /pattern/flags;
const regex2 = /hello/i;  // Case-insensitive
Best for: Static patterns known at compile time

Constructor Function

// Using RegExp constructor
const regex1 = new RegExp("pattern", "flags");
const regex2 = new RegExp("hello", "i");
Best for: Dynamic patterns built at runtime

Character Classes & Metacharacters

Basic Characters

Special metacharacters that have specific meanings in regex patterns.

. (Dot)

Matches: Any character except newline

const pattern = /a.c/;
console.log(pattern.test("abc")); // true
console.log(pattern.test("a1c")); // true
console.log(pattern.test("a@c")); // true
\ (Backslash)

Purpose: Escape character for special characters

const pattern = /\./;
console.log(pattern.test(".")); // true
console.log(pattern.test("a")); // false
| (Pipe)

Purpose: Alternation (OR operator)

const pattern = /cat|dog/;
console.log(pattern.test("I have a cat")); // true
console.log(pattern.test("I have a dog")); // true
() (Parentheses)

Purpose: Grouping and capturing

const pattern = /(abc)+/;
console.log(pattern.test("abc")); // true
console.log(pattern.test("abcabc")); // true
[] (Square Brackets)

Purpose: Character set (matches any character inside)

const pattern = /[abc]/;
console.log(pattern.test("a")); // true
console.log(pattern.test("d")); // false
[^] (Negated Set)

Purpose: Negated character set (matches anything NOT in the set)

const pattern = /[^abc]/;
console.log(pattern.test("d")); // true
console.log(pattern.test("a")); // false

Character Classes

Predefined shortcuts for common character sets.

\d (Digit)

Matches: Any digit (0-9)

const pattern = /\d{3}/;
console.log(pattern.test("123")); // true
console.log(pattern.test("abc")); // false
\D (Non-Digit)

Matches: Any non-digit character

const pattern = /\D+/;
console.log(pattern.test("abc")); // true
console.log(pattern.test("123")); // false
\w (Word Character)

Matches: Letters, digits, and underscore (a-z, A-Z, 0-9, _)

const pattern = /\w+/;
console.log(pattern.test("word123")); // true
console.log(pattern.test("test_var")); // true
\W (Non-Word Character)

Matches: Any non-word character

const pattern = /\W+/;
console.log(pattern.test("!@#")); // true
console.log(pattern.test("abc")); // false
\s (Whitespace)

Matches: Whitespace characters (space, tab, newline)

const pattern = /\s+/;
console.log(pattern.test(" ")); // true
console.log(pattern.test("\t")); // true
\S (Non-Whitespace)

Matches: Any non-whitespace character

const pattern = /\S+/;
console.log(pattern.test("word")); // true
console.log(pattern.test(" ")); // false

Quantifiers

Control how many times a pattern should match.

* (Asterisk)

Matches: Zero or more occurrences

const pattern = /ab*/;
console.log(pattern.test("a")); // true
console.log(pattern.test("abbb")); // true
+ (Plus)

Matches: One or more occurrences

const pattern = /ab+/;
console.log(pattern.test("ab")); // true
console.log(pattern.test("a")); // false
? (Question Mark)

Matches: Zero or one occurrence (optional)

const pattern = /colou?r/;
console.log(pattern.test("color")); // true
console.log(pattern.test("colour")); // true
{n} (Exact Count)

Matches: Exactly n occurrences

const pattern = /\d{3}/;
console.log(pattern.test("123")); // true
console.log(pattern.test("12")); // false
{n,} (Minimum Count)

Matches: n or more occurrences

const pattern = /\d{3,}/;
console.log(pattern.test("123")); // true
console.log(pattern.test("12345")); // true
{n,m} (Range Count)

Matches: Between n and m occurrences

const pattern = /\d{3,5}/;
console.log(pattern.test("123")); // true
console.log(pattern.test("123456")); // true

Anchors

Match positions rather than characters.

^ (Caret)

Matches: Start of string or line

const pattern = /^Hello/;
console.log(pattern.test("Hello world")); // true
console.log(pattern.test("Say Hello")); // false
$ (Dollar)

Matches: End of string or line

const pattern = /world$/;
console.log(pattern.test("Hello world")); // true
console.log(pattern.test("world Hello")); // false
\b (Word Boundary)

Matches: Position at word boundary

const pattern = /\bcat\b/;
console.log(pattern.test("cat")); // true
console.log(pattern.test("category")); // false
\B (Non-Word Boundary)

Matches: Position not at word boundary

const pattern = /\Bcat\B/;
console.log(pattern.test("category")); // true
console.log(pattern.test("cat")); // false

Common Regex Patterns

Here are some frequently used regex patterns for common validation and text processing tasks:

Validation Patterns

Common patterns for validating user input and data formats.

Email Address
const pattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
console.log(pattern.test("user@example.com")); // true
console.log(pattern.test("invalid.email")); // false

Validates: Basic email format with @ symbol and domain

Phone Number (US)
const pattern = /^(\+1[-.\s]?)?\(?([0-9]{3})\)?[-.\s]?([0-9]{3})[-.\s]?([0-9]{4})$/;
console.log(pattern.test("(555) 123-4567")); // true
console.log(pattern.test("555.123.4567")); // true

Validates: US phone numbers in various formats

Strong Password
const pattern = /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{8,}$/;
console.log(pattern.test("MyPass123!")); // true
console.log(pattern.test("password")); // false

Validates: At least 8 characters with lowercase, uppercase, digit, and special character

URL
const pattern = /^https?:\/\/(?:[-\w.])+(?:[:\d]+)?(?:\/(?:[\w/_.])*)?$/;
console.log(pattern.test("https://example.com")); // true
console.log(pattern.test("http://site.com:8080/path")); // true

Validates: HTTP and HTTPS URLs with optional ports and paths

Data Extraction

Patterns for extracting specific information from text.

Date Extraction
const pattern = /(\d{4})-(\d{2})-(\d{2})/g;
const text = "Events on 2023-12-25 and 2024-01-15";
console.log(text.match(pattern)); // ["2023-12-25", "2024-01-15"]

Extracts: Dates in ISO format (YYYY-MM-DD)

Hashtags
const pattern = /#\w+/g;
const text = "Learning #JavaScript and #RegEx #coding";
console.log(text.match(pattern)); // ["#JavaScript", "#RegEx", "#coding"]

Extracts: All hashtags from text

Currency Amounts
const pattern = /\$\d{1,3}(?:,\d{3})*(?:\.\d{2})?/g;
const text = "Prices: $29.99, $1,234.56, $10,000";
console.log(text.match(pattern)); // ["$29.99", "$1,234.56", "$10,000"]

Extracts: Dollar amounts with proper formatting

HTML Tags
const pattern = /<([a-z]+)(\s[^>]*)?>.*?<\/\1>/gi;
const html = "
Content
 
Text
";
console.log(html.match(pattern)); // ["
Content
", "
Text
"]

Extracts: HTML tags with their content

Text Formatting

Transform and clean text using replacement patterns.

Title Case
const text = "hello world javascript";
const titleCase = text.replace(/\b\w/g, char => char.toUpperCase());
console.log(titleCase); // "Hello World Javascript"

Converts: First letter of each word to uppercase

Remove Extra Spaces
const text = "text   with    extra     spaces";
const cleaned = text.replace(/\s+/g, " ");
console.log(cleaned); // "text with extra spaces"

Removes: Multiple consecutive spaces

Format Phone Numbers
const phone = "1234567890";
const formatted = phone.replace(/(\d{3})(\d{3})(\d{4})/, "($1) $2-$3");
console.log(formatted); // "(123) 456-7890"

Formats: 10-digit numbers into (XXX) XXX-XXXX

Mask Sensitive Data
const card = "1234567890123456";
const masked = card.replace(/(\d{4})\d{8}(\d{4})/, "$1********$2");
console.log(masked); // "1234********3456"

Masks: Middle digits of credit card numbers

JavaScript Regex Methods

JavaScript provides several methods for working with regular expressions:

test()

Returns: boolean

Tests if a pattern matches in a string. Returns true or false. 

const regex = /hello/i;
console.log(regex.test("Hello World")); // true
console.log(regex.test("Hi there"));    // false
Result: true

match()

Returns: Array | null

Retrieves the matches when matching a string against a regex. 

const text = "The year 2023 was great!";
const matches = text.match(/\d+/g);
console.log(matches); // ["2023"]
Result: ["5", "3"]

replace()

Returns: string

Returns a new string with matches replaced by a replacement string. 

const text = "Hello World";
const newText = text.replace(/World/, "JavaScript");
console.log(newText); // "Hello JavaScript"
Result: "Hello JavaScript! JavaScript is great!"

search()

Returns: number

Returns the index of the first match, or -1 if not found. 

const text = "Find the number 42 here";
const index = text.search(/\d+/);
console.log(index); // 16
Result: 14

split()

Returns: Array

Splits a string into an array using a regex as the delimiter. 

const text = "apple,banana;orange:grape";
const fruits = text.split(/[,;:]/);
console.log(fruits); // ["apple", "banana", "orange", "grape"]
Result: ["apple", "banana", "orange", "grape"]

exec()

Returns: Array | null

Executes a search for a match and returns detailed information. 

const regex = /(\d{4})-(\d{2})-(\d{2})/;
const result = regex.exec("Today is 2023-12-25");
console.log(result); // ["2023-12-25", "2023", "12", "25", ...]
Result: ["2023-12-25", "2023", "12", "25"]
