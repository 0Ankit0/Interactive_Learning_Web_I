What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure and content of a web page using elements and tags.

HTML Document Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to HTML!</h1>
    <p>This is my first web page.</p>
  </body>
</html>

Evolution of HTML

1990

HTML Birth

Tim Berners-Lee creates the first version of HTML as part of the World Wide Web project at CERN.

1995

HTML 2.0

First standardized version with forms, tables, and image support.

1997

HTML 4.0

Introduces stylesheets (CSS), scripting, and improved accessibility features.

2014

HTML5

Modern standard with semantic elements, multimedia support, and enhanced APIs.

Essential HTML Tags

<h1> - <h6>

Heading tags for creating hierarchical content structure

<h1>Main Title</h1>

<p>

Paragraph tag for text content

<p>This is a paragraph.</p>

<a>

Anchor tag for creating hyperlinks

<a href="https://example.com">Link</a>

<img>

Image tag for displaying pictures

<img src="image.jpg" alt="Description">

<div>

Division tag for grouping content

<div class="container">Content</div>

<span>

Inline element for styling parts of text

<span style="color: red;">Text</span>
