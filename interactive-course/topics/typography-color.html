Type Anatomy

Understanding the anatomy of typefaces helps you make informed decisions about font selection and text treatment.

Typography
Baseline: The imaginary line where letters sit
X-Height: Height of lowercase letters like 'x'
Cap Height: Height of capital letters
Ascender: Parts that extend above x-height (b, h, k)
Descender: Parts that extend below baseline (g, p, y)

Typography CSS Best Practices

/* CSS Custom Properties for Typography */
:root {
  /* Font Families */
  --font-primary: 'Inter', sans-serif;
  --font-heading: 'Playfair Display', serif;
  --font-mono: 'Fira Code', monospace;
  
  /* Font Sizes - Modular Scale */
  --text-xs: 0.75rem;    /* 12px */
  --text-sm: 0.875rem;   /* 14px */
  --text-base: 1rem;     /* 16px */
  --text-lg: 1.125rem;   /* 18px */
  --text-xl: 1.25rem;    /* 20px */
  --text-2xl: 1.5rem;    /* 24px */
  --text-3xl: 1.875rem;  /* 30px */
  --text-4xl: 2.25rem;   /* 36px */
  
  /* Line Heights */
  --leading-tight: 1.25;
  --leading-normal: 1.5;
  --leading-relaxed: 1.75;
  
  /* Letter Spacing */
  --tracking-tight: -0.025em;
  --tracking-normal: 0;
  --tracking-wide: 0.025em;
}

/* Typography Classes */
.text-heading {
  font-family: var(--font-heading);
  font-weight: 700;
  line-height: var(--leading-tight);
  letter-spacing: var(--tracking-tight);
}

.text-body {
  font-family: var(--font-primary);
  font-weight: 400;
  line-height: var(--leading-normal);
  letter-spacing: var(--tracking-normal);
}

/* Responsive Typography */
h1 {
  font-size: clamp(var(--text-2xl), 4vw, var(--text-4xl));
}

/* Improve Readability */
p {
  max-width: 65ch; /* Optimal line length */
  margin-bottom: 1.5em;
}

/* Web Font Loading */
@font-face {
  font-family: 'Inter';
  src: url('inter.woff2') format('woff2');
  font-display: swap; /* Improve loading performance */
}

  --color-gray-50: hsl(210, 20%, 98%);
  --color-gray-100: hsl(210, 16%, 93%);
  --color-gray-300: hsl(210, 14%, 83%);
  --color-gray-500: hsl(210, 11%, 71%);
  --color-gray-700: hsl(210, 10%, 40%);
  --color-gray-900: hsl(210, 24%, 16%);
  
  /* Semantic Colors */

/* Dark Mode Support */
@media (prefers-color-scheme: dark) {
  :root {
    --color-text-primary: var(--color-gray-50);
    --color-text-secondary: var(--color-gray-300);
    --color-text-muted: var(--color-gray-500);
  }
}

/* High Contrast Mode Support */
@media (prefers-contrast: high) {
  :root {
    --color-primary: hsl(210, 100%, 30%);
    --color-text-primary: hsl(0, 0%, 0%);
  }
}

/* Reduced Motion Support */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}

/* Print Styles */
@media print {
  :root {
    --color-primary: hsl(0, 0%, 0%);
    --color-text-primary: hsl(0, 0%, 0%);
  }
  
  .bg-primary {
    background: transparent !important;
    border: 1px solid black;
  }
}
