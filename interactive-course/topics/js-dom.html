What is the DOM?

The Document Object Model (DOM) is a programming interface that represents the structure of HTML and XML documents as a tree of objects. It allows JavaScript to:

  • Access Elements: Find and select HTML elements
  • Modify Content: Change text, HTML, and attributes
  • Change Styles: Modify CSS properties dynamically
  • Add/Remove Elements: Create new elements or delete existing ones
  • Respond to Events: Handle user interactions

Key Concept:

The DOM treats your HTML document as a tree structure where every element, attribute, and text is a node that can be accessed and modified with JavaScript.

DOM Selectors

DOM Selection Methods

getElementById()

Select an element by its ID (returns single element) 

// HTML: <div id="myDiv">Hello</div>
const element = document.getElementById('myDiv');
console.log(element.textContent); // "Hello"

getElementsByClassName()

Select elements by class name (returns HTMLCollection) 

// HTML: <p class="highlight">Text</p>
const elements = document.getElementsByClassName('highlight');
console.log(elements.length); // Number of elements

getElementsByTagName()

Select elements by tag name (returns HTMLCollection) 

// Select all paragraphs
const paragraphs = document.getElementsByTagName('p');
for (let p of paragraphs) {
  console.log(p.textContent);
}

Query Methods

querySelector()

Select first element matching CSS selector

// Select first element with class 'button'
const button = document.querySelector('.button');
// Select element by ID
const header = document.querySelector('#header');

querySelectorAll()

Select all elements matching CSS selector (returns NodeList) 

// Select all buttons
const buttons = document.querySelectorAll('button');
// Select all elements with class 'card'
const cards = document.querySelectorAll('.card');

Modifying Content

Content Modification Properties

textContent

Gets or sets the text content (no HTML)

const element = document.getElementById('myDiv');
element.textContent = 'New text content';
console.log(element.textContent); // "New text content"

innerHTML

Gets or sets HTML content inside element

const element = document.getElementById('myDiv');
element.innerHTML = '<strong>Bold text</strong>';
// Renders as bold text in the browser

outerHTML

Gets or sets the entire element including its tags

const element = document.getElementById('myDiv');
console.log(element.outerHTML);
// "<div id='myDiv'>content</div>"

value

Gets or sets the value of form elements

const input = document.getElementById('myInput');
input.value = 'New input value';
console.log(input.value); // "New input value"

Modifying Styles

Style Modification Techniques

Direct Style Property

const element = document.getElementById('myDiv');
element.style.backgroundColor = 'blue';
element.style.color = 'white';
element.style.fontSize = '18px';
element.style.display = 'none'; // Hide element

CSS Text Property

const element = document.getElementById('myDiv');
element.style.cssText = 'color: red; background: yellow; padding: 10px;';
// Or
element.setAttribute('style', 'color: red; background: yellow;');

Class Manipulation

const element = document.getElementById('myDiv');
element.classList.add('highlight');
element.classList.remove('old-style');
element.classList.toggle('active');
element.classList.contains('highlight'); // true/false

Working with Attributes

Attribute Manipulation Methods

getAttribute() / setAttribute()

const link = document.getElementById('myLink');

// Get attribute
const href = link.getAttribute('href');
console.log(href); // "https://example.com"

// Set attribute
link.setAttribute('href', 'https://newsite.com');
link.setAttribute('target', '_blank');

hasAttribute() / removeAttribute()

const img = document.getElementById('myImage');

// Check if attribute exists
if (img.hasAttribute('alt')) {
  console.log('Image has alt text');
}

// Remove attribute
img.removeAttribute('title');

Data Attributes

// HTML: <div data-user-id="123" data-role="admin"></div>
const element = document.getElementById('myDiv');

// Access data attributes
console.log(element.dataset.userId); // "123"
console.log(element.dataset.role);   // "admin"

// Set data attributes
element.dataset.status = 'active';

Event Handling

Event Handling Methods

addEventListener()

Attach an event handler to an element

const button = document.getElementById('myButton');
button.addEventListener('click', function() {
  console.log('Button was clicked!');
});

// Or with arrow function
button.addEventListener('click', () => {
  alert('Button clicked!');
});

Event Object

Access event information and prevent default behavior

button.addEventListener('click', function(event) {
  event.preventDefault(); // Prevent default action
  console.log('Event type:', event.type);
  console.log('Target element:', event.target);
});

Common Events

Mouse, keyboard, and form events

// Mouse events
element.addEventListener('click', handler);
element.addEventListener('mouseover', handler);
element.addEventListener('mouseout', handler);

// Keyboard events
input.addEventListener('keydown', handler);
input.addEventListener('keyup', handler);

// Form events
form.addEventListener('submit', handler);
input.addEventListener('change', handler);

Practical Examples

Dynamic To-Do List

Create an interactive to-do list with add/remove functionality. 

// HTML: <input id="todo-input"> <button id="add-btn">Add</button> <ul id="todo-list"></ul>

const input = document.getElementById('todo-input');
const addBtn = document.getElementById('add-btn');
const todoList = document.getElementById('todo-list');

addBtn.addEventListener('click', function() {
  const todoText = input.value.trim();
  if (todoText) {
    const li = document.createElement('li');
    li.innerHTML = `${todoText} <button class="delete-btn">×</button>`;
    todoList.appendChild(li);
    input.value = '';

    // Add delete functionality
    li.querySelector('.delete-btn').addEventListener('click', function() {
      li.remove();
    });
  }
});

Image Gallery with Lightbox

Create a responsive image gallery with modal lightbox functionality. 

// Select all gallery images
const images = document.querySelectorAll('.gallery img');
const modal = document.getElementById('modal');
const modalImg = document.getElementById('modal-image');

images.forEach(img => {
  img.addEventListener('click', function() {
    modal.style.display = 'block';
    modalImg.src = this.src;
  });
});

// Close modal when clicking outside
modal.addEventListener('click', function() {
  modal.style.display = 'none';
});

Form Validation

Real-time form validation with visual feedback.

const emailInput = document.getElementById('email');
const submitBtn = document.getElementById('submit-btn');

emailInput.addEventListener('input', function() {
  const email = this.value;
  const isValid = /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);

  if (isValid) {
    this.classList.remove('invalid');
    this.classList.add('valid');
    submitBtn.disabled = false;
  } else {
    this.classList.remove('valid');
    this.classList.add('invalid');
    submitBtn.disabled = true;
  }
});

Accordion Component

Create collapsible accordion sections.

const accordionHeaders = document.querySelectorAll('.accordion-header');

accordionHeaders.forEach(header => {
  header.addEventListener('click', function() {
    // Close other accordions
    accordionHeaders.forEach(h => {
      if (h !== this) {
        h.classList.remove('active');
        h.nextElementSibling.style.display = 'none';
      }
    });

    // Toggle current accordion
    this.classList.toggle('active');
    const content = this.nextElementSibling;
    content.style.display = content.style.display === 'block' ? 'none' : 'block';
  });
});
