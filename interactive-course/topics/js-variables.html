Introduction to JavaScript Variables & Data Types

What are Variables?

Variables are containers that store data values. In JavaScript, variables can hold different types of data and can be reassigned with new values during program execution.

What are Data Types?

Data types specify what kind of data can be stored in a variable. JavaScript has dynamic typing, meaning variables can hold different types of values throughout their lifecycle.

Quick Examples:


// var - function scoped
var name = "John";
var name = "Jane"; // Allowed (redeclaration)
name = "Bob"; // Allowed (reassignment)

// let - block scoped
let age = 25;
// let age = 30; // Error: Cannot redeclare
age = 30; // Allowed (reassignment)

// const - block scoped, immutable
const PI = 3.14159;
// PI = 3.14; // Error: Cannot reassign
// const E; // Error: Must initialize

Variable Scope

Understanding where variables can be accessed in your code is crucial for writing effective JavaScript.

Global Scope

Variables declared outside any function or block have global scope and can be accessed from anywhere in the code.


var globalVar = "I'm global";
let globalLet = "Me too";

function test() {
  console.log(globalVar); // Accessible
  console.log(globalLet); // Accessible
}

Function Scope

Variables declared with var inside a function are only accessible within that function.


function myFunction() {
  var functionScoped = "Only in function";
  let alsoFunctionScoped = "Me too";
  
  console.log(functionScoped); // Works
}

// console.log(functionScoped); // Error!

Block Scope

Variables declared with let and const inside a block {} are only accessible within that block.


if (true) {
  var varExample = "var ignores blocks";
  let letExample = "let respects blocks";
  const constExample = "const respects blocks";
}

console.log(varExample); // Works
// console.log(letExample); // Error!
// console.log(constExample); // Error!

JavaScript Data Types

JavaScript has dynamic typing, meaning variables can hold different types of values.

Number

Represents both integers and floating-point numbers.


let age = 25;          // Integer
let price = 19.99;     // Float
let negative = -42;    // Negative
let infinity = Infinity; // Special value
let notNumber = NaN;   // Not a Number

String

Represents textual data enclosed in quotes.


let firstName = "John";  // Double quotes
let lastName = 'Doe';    // Single quotes
let fullName = `${firstName} ${lastName}`; // Template literal
let multiline = `Line 1
Line 2`; // Multiline string

Boolean

Represents logical values: true or false.


let isActive = true;
let isComplete = false;
let hasAccess = Boolean(1);  // true
let isEmpty = Boolean(0);    // false
let result = 5 > 3;         // true

Undefined

Variable declared but not assigned a value.


let x;                     // undefined
let y = undefined;         // explicitly undefined
function test() {}         // returns undefined
let obj = {};
console.log(obj.missing);  // undefined

Null

Represents intentional absence of any object value.


let data = null;           // intentionally empty
let response = null;       // no data received

// Note: typeof null === "object" (JavaScript quirk)
console.log(null == undefined);  // true
console.log(null === undefined); // false

Object

Complex data type for collections and entities.


let person = {             // Object literal
  name: "Alice",
  age: 30
};
let numbers = [1, 2, 3];   // Array (special object)
let today = new Date();    // Date object
let pattern = /abc/g;      // RegExp object

Type Conversion

JavaScript can automatically convert between types or you can convert explicitly.

Automatic Type Conversion (Coercion)


// Automatic conversions
console.log("5" + 3);      // "53" (number to string)
console.log("5" - 3);      // 2 (string to number)
console.log("5" * "2");    // 10 (both to numbers)
console.log(true + 1);     // 2 (boolean to number)
console.log(false + 1);    // 1 (boolean to number)
console.log("" + 42);      // "42" (number to string)
console.log(Boolean(0));   // false
console.log(Boolean(1));   // true

Explicit Type Conversion


// String conversion
let num = 123;
let str = String(num);     // "123"
let str2 = num.toString(); // "123"

// Number conversion
let text = "456";
let number = Number(text); // 456
let int = parseInt(text);  // 456
let float = parseFloat("3.14"); // 3.14

// Boolean conversion
let bool1 = Boolean(1);    // true
let bool2 = Boolean(0);    // false
let bool3 = Boolean("");   // false
let bool4 = Boolean("hi"); // true
