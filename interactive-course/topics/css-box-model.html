What is the CSS Box Model?

The CSS Box Model is a fundamental concept that describes how elements are structured and sized in web layouts. Every HTML element is essentially a rectangular box consisting of four main components:

  • Content: The actual content of the element (text, images, etc.)
  • Padding: Space between content and border (inside the element)
  • Border: A border surrounding the padding (and content)
  • Margin: Space outside the border (between elements)

Key Insight:

Understanding the box model is crucial for controlling spacing, alignment, and the overall layout of web pages.

Box Model Properties

Content

The innermost part containing text, images, or other media. 

width: 200px;
height: 100px;

Padding

Space between content and border. Creates internal breathing room. 

padding: 20px; /* All sides */
padding: 10px 20px; /* Top/bottom, left/right */
padding: 10px 15px 20px 25px; /* T, R, B, L */

Border

A visible or invisible line around the padding and content. 

border: 2px solid #333;
border-width: 1px;
border-style: solid;
border-color: blue;

Margin

Space outside the border. Creates distance between elements. 

margin: 20px; /* All sides */
margin: 10px auto; /* Top/bottom, auto left/right */
margin-top: 10px;

Box Sizing Property

The box-sizing property controls how the total width and height of an element is calculated:

Content-Box (Default):

Width = content only

Total: 200px + 20px + 5px = 225px

Border-Box:

Width = total box size

Total: 200px (includes padding & border)

Content-Box (Default):

/* Default behavior */
box-sizing: content-box;
width: 200px;
padding: 20px;
border: 5px solid #ccc;
/* Total width: 200 + 40 + 10 = 250px */

Border-Box (Recommended):

/* More intuitive */
box-sizing: border-box;
width: 200px;
padding: 20px;
border: 5px solid #ccc;
/* Total width: exactly 200px */

Centering with Margins

Auto Margins for Horizontal Centering:

I'm horizontally centered!

.centered-box {
  width: 300px;
  margin: 0 auto; /* Top/bottom: 0, Left/right: auto */
  background: #e3f2fd;
  padding: 20px;
  text-align: center;
}

Centering Techniques:

  • Horizontal: margin: 0 auto; (requires fixed width)
  • Text/Inline elements: text-align: center;
  • Flexbox: justify-content: center; align-items: center;
