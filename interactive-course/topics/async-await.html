What is Async/Await?

Async/await is a modern JavaScript feature for writing asynchronous code that looks and behaves like synchronous code. It is built on top of Promises and makes it easier to work with asynchronous operations such as API calls, timers, and file I/O.

Async Functions

Declare functions as async to use await inside them.

async function fetchData() {
  const response = await fetch('https://api.example.com/data');
  const data = await response.json();
  return data;
}

Await Keyword

Pause execution until a Promise resolves, making code easier to read. 

const result = await someAsyncOperation();
console.log(result);

Error Handling

Use try/catch blocks for robust error handling in async code. 

try {
  await doSomethingAsync();
} catch (error) {
  console.error(error);
}

Promise Integration

Async/await works seamlessly with existing Promise-based APIs. 

async function getUser() {
  return await fetchUserFromAPI();
}
